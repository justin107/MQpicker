**业务场景**

多个单选
或多个单选加一（单、多选）自定义
或者自定义一（单、多选）
例如：
常规省市区三级级
或省市区街道自定义五级
或市区自定义三级
或市区街道自定义四级
等等，符合大多数业务场景
统一接口参数也可以全自定义无限拓展

JQ，高德js API，已有微调即可
https://lbs.amap.com/api/javascript-api/summary

1、业务产出districtCode

2、支持多实例

4、支持省-直辖市字母排序

5、支持浮窗显示

6、支持省市自定义区域

7、支持单选、多选、全选

8、单级自定义数据

9、支持级别调整
